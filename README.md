# Valentine React App

Милое интерактивное React-приложение с вопросом «Будешь моей валентинкой?».

## Запуск локально

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```

## Деплой на Vercel

1. Импортируй репозиторий в Vercel.
2. Vercel автоматически определит Vite через `vercel.json`.
3. Build command: `npm run build`
4. Output directory: `dist`
